# Drive Game Demo

A standalone Cargo project that demonstrates the Solite source workflow:

- debug builds mount `ui/` directly and live-reload TSX/CSS changes
- release builds mount the bundled `OUT_DIR` output generated by `build.rs`
- WebGPU renders the driving scene behind a fullscreen Solite HUD

Run from the repository root:

```sh
cargo run -p solite-drive-game
```

Run the release bundle path:

```sh
cargo run -p solite-drive-game --release
```

Controls:

- Up Arrow: throttle
- Down Arrow: brake, then reverse once stopped
- Left Arrow / Right Arrow: steer
- Escape: quit

Drag the vertical slider in the top-right of the HUD to set the maximum speed
(20–50 mph). The slider value is sent back to the Rust simulation via
`sendEvent`, which clamps the car's top speed accordingly.
